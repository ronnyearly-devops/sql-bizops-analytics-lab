# SQL BizOps Analytics Lab

## Overview

This project demonstrates SQL-based business operations analytics using a simulated customer, order, payment, and support ticket database.

The objective was to analyze operational metrics, customer activity, revenue performance, payment processing, and support operations using SQL queries commonly used by Business Operations, Operations Engineering, and Analytics teams.

## Technologies Used

* SQLite
* SQL
* Business Operations Analytics
* KPI Reporting
* Relational Databases

## Database Tables

### customers

Stores customer information and account details.

### orders

Stores customer order activity and transaction records.

### payments

Tracks payment processing status and payment metrics.

### support_tickets

Tracks customer support interactions and operational issues.
